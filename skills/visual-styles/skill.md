# Visual Styles Skill

## Button Design
- Buttons should have both icons and short explanatory names for clarity and accessibility.
- For button groups in popup panels, prefer square buttons arranged in grids for a clean, modern look and easy touch interaction.
- Avoid icon-only buttons unless the meaning is universally clear or a tooltip is provided.
- On mobile, ensure button labels remain visible or easily discoverable.

## Lessons Learned
- Icon-only buttons can be unclear, especially on mobile. Always pair icons with concise text labels.
- Use tooltips for additional explanation if space is limited.
